# Web Development Project 6 - *WeatherDashboard*

Submitted by: **Tony Wu**

This web app: **A weather app that displays daily tempature with precipitation in inches in NY, shows the highest and lowest degrees in Celsius. Sunrise, Sunset, and has two filter, one for filtering by time of day, another for filtering by precipitation. With a chart that displays the weather, and clickable data types and links on the day of the weather**

Time spent: **6** hours spent in total
